Acerca de mí

¿Qué estoy haciendo actualmente?

Llevo a cabo tareas con el rol de fullstack developer utilizando Java como lenguaje de backend, MySQL y LocalStorage como lenguaje de bases de datos, y JavaScript + HTML (con Thymeleaf) + CSS para el frontend.

Mi formación

Ingeniería en Petróleo

Mi carrera se refiere al diseño, la implantación y el funcionamiento de procesos técnicos centrados en la exploración y la producción de crudo y gas natural. Los ingenieros de este sector se involucran en el proceso de extracción de estos recursos para determinar los mejores métodos de perforación y monitorizar las operaciones diarias.

Programador Web FullStack

He aprendido a crear aplicaciones web full stack, con diseño front-end (HTML + CSS, Bootstrap, Thymeleaf) y arquitectura back-end (Programación orientada a objetos, Java, MySQL, Spring Boot). El trabajo en equipo es la interacción y sentido de unidad entre personas que colaboran con una prioridad común para alcanzar objetivos que difícilmente se lograrían de forma individual. De esta forma es la educación recibida por parte de EGG.
